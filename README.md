## Overview

DevLoop is an AI-powered Software Engineering Management Platform designed to automate and optimize the Software Development Life Cycle (SDLC).

The platform combines project management, team collaboration, sprint planning, task tracking, software health monitoring, analytics, and AI-powered engineering agents into a single unified dashboard.

DevLoop helps engineering teams improve productivity, monitor project risks, manage resources efficiently, and accelerate software delivery.

## Role-Based Access Control

DevLoop supports multiple engineering roles:

| Role | Access |
|--------|---------|
| ADMIN | Complete system access |
| PROJECT_MANAGER | Projects, Team, Analytics, Health |
| DEVELOPER | Tasks, Agents, Dashboard |
| TESTER | Tasks, Agents, Dashboard |
| DEVOPS | Agents, Infrastructure Operations |

The platform uses JWT-based authentication and role-based authorization for secure access management.
